About Shreya's Kitchen

A home kitchen, run the way your grandmother would run it.

Shreya's Kitchen started with a simple idea: not everyone has the time to roll fresh rotis every day, but everyone deserves to eat like they're at home. Every roti, bhakri and poli here is hand-rolled in small batches, using the same flour, the same patience, and the same care as a family kitchen.

"No shortcuts. No preservatives. No machine-pressed dough."
